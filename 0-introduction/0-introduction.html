第一讲 课程介绍

1. 课程概要

在代码实现层面学习Web应用开发的前端网页设计、后端服务搭建、数据读写等

从技术实现角度了解信息系统的运作模式

2. 课程目标

掌握基本的静态网页设计

理解动态网页的前端实现技术

了解Web服务的后端实现技术

能够实现简单的全栈Web应用开发

3. 授课方式

课堂知识介绍、实时写代码、课堂练习

以技术点为主的编码为辅

系统性的web应用开发为主

4. 考核方式

课堂考勤 10%

课堂/课后、期中联系 40%

课程作业 50%（1-4人分组选题、或参与开源项目）
